# Jason Carrasco - Marketing VP Portfolio

A professional, responsive portfolio website showcasing Jason Carrasco's marketing expertise and prestigious client work.

## Features

- **Modern Design**: Clean, professional layout with gradient backgrounds and smooth animations
- **Responsive Layout**: Fully responsive design that works on all devices
- **Interactive Navigation**: Smooth scrolling navigation with fixed header
- **Client Showcase**: Displays prestigious client logos and categories
- **Case Studies**: Featured case studies with hover effects
- **About Section**: Professional background with skills and statistics
- **Contact Information**: Footer with multiple contact methods

## Structure

### Navigation
- **Left Side**: "Jason Carrasco" logo linking to homepage
- **Right Side**: "About Me" link for navigation

### Sections
1. **Hero Section**: Eye-catching introduction with call-to-action buttons
2. **Client Showcase**: Grid of client categories (Fortune 500, Tech Giants, etc.)
3. **Case Studies**: Three featured case studies with icons and descriptions
4. **About Section**: Professional background, skills, and statistics
5. **Footer**: Contact information and social links

## Files

- `index.html` - Main HTML structure
- `styles.css` - Complete CSS styling with responsive design
- `script.js` - JavaScript for interactive features
- `README.md` - This documentation file

## Technologies Used

- **HTML5**: Semantic markup

- **JavaScript**: Interactive features and smooth scrolling

## Customization

### Colors
The website uses a professional color palette:
- Primary Blue: `#3b82f6`
- Dark Gray: `#1f2937`
- Light Gray: `#f9fafb`
- Text Gray: `#6b7280`

### Content
To customize the content:
1. Update the hero section text in `index.html`
2. Modify client categories in the client showcase
3. Edit case study content and descriptions
4. Update contact information in the footer
5. Adjust statistics in the about section

### Styling
To modify the design:
1. Update colors in `styles.css`
2. Adjust spacing and typography
3. Modify animations and transitions
4. Customize responsive breakpoints
